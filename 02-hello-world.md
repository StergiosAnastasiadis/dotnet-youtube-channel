# Hello World!

We can use Visual Studio, Visual Studio Code, JetBrains Rider or Notepad.

We are going to use Visual Studio Code with the C# Dev Kit.

Now dotnet is a CLI used to create projects, run applications, do everythings you want to do with .NET
We have one entry point CLI
One entry point application

We will write 
dotnet new

We are going to create a console application, what is called a text mode application.
dotnet new console

Now we got some files with that

Program.cs is our main program, our entry point.

the .csproj and obj are supportive files.

To run
dotnet run

This will compile the application create the output and the run Hello World.

Now .NET and C# in this case is a compiled application.
So the Program.cs is turned into something that can run execute.

If we look at the folder again we can see an extra folder called bin or binary.
