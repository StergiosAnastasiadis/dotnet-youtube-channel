# Basics of VS Code and the C# DevKit

We will install the C# DevKit
And with that comes all these other tools to run Debug C# to get Intellisense.
Those make it easier to code C#

Without those VSCode would be like notepad for C#

It makes syntax highlighting
Also we get additional information from VS Code, on the left we have a Exploer.

We get a Solution Exploret with C# DevKit
This is not a physical view of what is inside the disk
It has some extra concepts and all ge got from files is the Program.cs.

It hid the bin and the object folder/files, Output folders, output files, intermidieate files.
It is showing the logical view of our project, our main source files, dependencies etc.

Just like a Dashboard on a car which is hiding all the details of the engine.

On the left we can find a button which says Run & Debug
This also outputs some exta stuff which will be usefull later.
Those are on the Debug Console, we can always click on Console tab.
We can do this debug with CTRL+F5
